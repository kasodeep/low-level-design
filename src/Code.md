# Clean Code

1. Have meaningful names to avoid extra comments.
2. Add comments to complex explanation or code that is not obvious.
3. Keep functions small, one responsibility and make it pure.
4. Command Query Separation (CQS) is a way of writing a function to change the state and query it.
